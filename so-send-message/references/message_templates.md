# 消息模板库

## 概述

本文档提供各种场景下的消息模板，便于快速构建高质量的群消息。

## 通用模板

### 1. 通知类模板

#### 简单通知
```markdown
📢 **通知**

{通知内容}

⏰ 时间: {时间}
📍 相关: {相关系统/人员}
```

#### 重要通知
```markdown
🚨 **重要通知**

{通知内容}

🔔 **影响范围**: {影响范围}
🕐 **持续时间**: {开始时间} - {结束时间}
```

## 模板使用说明

### 变量替换
模板中的`{变量名}`需要替换为实际值：

```python
template = """📢 **通知**

{通知内容}

⏰ 时间: {时间}
"""

message = template.format(
    通知内容="系统将于今晚进行维护",
    时间="2024-01-15 22:00-24:00"
)
```

### 动态生成
可以编写函数动态生成消息：

```python
def generate_alert_message(system, level, content):
    icons = {
        "P0": "🔴",
        "P1": "🟠",
        "P2": "🟡",
        "P3": "🟢"
    }
    
    return f"""{icons.get(level, '⚠️')} **系统告警**

**系统**: {system}
**级别**: {level}
**时间**: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}
**内容**: {content}
"""
```

### 组合使用
可以将多个模板组合使用：

```python
def generate_daily_report(systems):
    # 生成系统状态表格
    table_rows = []
    for system in systems:
        table_rows.append(f"| {system['name']} | {system['status']} | {system['availability']} |")
    
    table = "\n".join(table_rows)
    
    # 使用模板
    return f"""📊 **监控日报**

**日期**: {datetime.now().strftime('%Y-%m-%d')}

**系统状态**:
```
| 系统 | 状态 | 可用性 |
{table}
```
"""
```

## 最佳实践

1. **使用恰当的图标**：图标能快速传达消息类型和紧急程度
2. **保持结构清晰**：使用标题、列表、表格等格式化元素
3. **突出重点信息**：使用加粗、颜色等强调关键信息
4. **控制消息长度**：过长的消息考虑分多次发送或使用折叠
5. **定期更新模板**：根据反馈和使用情况优化模板
6. **统一风格**：在团队或组织内保持消息风格一致
