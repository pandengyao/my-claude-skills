# 百度Ai+短剧创作平台全流程操作技能手册

## 一、基础入门必备

### 入口与登录规范

- **平台入口**：https://shenghua.baidu.com/videoplatform/home
- **支持端**：仅PC端操作
- **登录方式**：百度APP扫码、手机号短信、账号密码三种模式
- **登录前**：需确认同意平台用户协议与隐私政策

### 核心能力认知

平台支持「剧本→成片」全流程自动化创作，同时开放全链路人机协同编辑能力，可对角色、分镜、画面、视频、音频、字幕等全维度自定义调整。

**核心创作链路（固定顺序）**：

```
上传小说/剧本 → 剧本大纲 → 角色场景 → 分镜规划 → 分镜画面 → 分镜视频 → 自动剪辑合成
```

## 二、全流程核心创作操作

### 步骤1：剧本导入环节

#### 格式合规前置处理

| 文件类型 | 支持格式 |
|---------|---------|
| 小说 | txt |
| 剧本 | docx、txt |

⚠️ 提前按规范整理文件，避免上传失败；可下载平台官方样例参考格式规范。

#### 双创作路径选择

| 路径 | 说明 |
|-----|------|
| 原创内容 | 直接本地上传自制剧本/小说，一键启动创作流程 |
| IP改编 | 在平台小说库中，按频道、类型、字数精准筛选优质IP，点击「去创作」直接启动，省去剧本改编成本 |

#### 创作参数预设技巧

上传前提前选定核心参数，减少后续返工成本：

| 参数类型 | 选型建议 |
|---------|---------|
| 剧本模型 | 剧本1.0：仅支持第三人称演绎，创意度更高；剧本2.0：支持第一人称解说+第三人称演绎，效果更稳定 |
| 短剧类型 | 第三人称短剧：适合常规剧情演绎；第一人称短剧：适合沉浸式、强代入感剧情 |
| 视频比例 | 竖屏9:16（短剧主流适配）、竖屏3:4、横屏3:2、横屏4:3按需选择 |
| 拆集规则 | 自动拆集：AI按剧情节奏自动拆分；手动拆集：支持1-10集自定义设置 |
| 单集时长 | 智能时长（AI自动适配）、1分钟、2分钟三档可选 |
| 画面风格 | 内置男频玄幻、霸总言情、惊悚悬疑等十余种风格，也支持自定义prompt精准设定 |

### 步骤2：剧本大纲环节

#### 角色清单精细化管理

AI自动生成全剧角色清单后，可补充：

- **角色指令词**：精准描述外貌、穿着、神态、性格
- **角色简介**：锁定人物核心设定

支持点击「添加角色」自定义新增人物：
- 可同步上传角色参考图
- 可从角色库选用形象
- 提前保证全片人物一致性

⚠️ **关键提醒**：角色生成后，剧本大纲不支持回退修改，需确认所有内容无误后，再点击「编辑角色场景」进入下一环节。

#### 剧集梗概节奏打磨

- 可直接编辑分集剧情、台词内容、画面标注、音效提示
- 精准卡点每一集的钩子和戏剧节奏
- 角色生成过程中可退出页面，在「作品管理」界面查看生成进度

### 步骤3：角色场景环节

#### 角色形象高效创作技巧

三种角色新增方式：

| 方式 | 说明 | 适用场景 |
|-----|------|---------|
| 精准prompt生成 | 推荐格式「时代背景+年龄+性别+外貌特征+穿着+神态+画面风格」 | 自定义角色 |
| 本地上传 | 直接上传定制好的角色形象图 | 已有角色设计 |
| 形象库选用 | 从公开形象库/个人形象库直接选用成熟角色 | 快速启动 |

**角色prompt示例**：
```
中国现代动漫风格，20岁男性，黑色短发，五官清秀，穿白色校服衬衫，眼神专注
```

**角色配音前置匹配**：
- 生成角色时同步选择适配人设的音色
- 点击声波图标可试听
- 支持后续随时更换

#### 场景定制核心技巧

**场景prompt推荐格式**：
```
画面风格 + 地点 + 环境特征 + 天气 + 景物细节
```

**场景prompt示例**：
```
中国现代动漫风格，偏远山区的古老宅院，大气但难掩破败，周围植被杂乱，阴雨天
```

- 支持上传参考图、prompt重绘场景
- 单集出镜角色与场景可灵活调整
- 历史生成记录支持回溯复用

### 步骤4：分镜规划环节

#### 双生产模式选型

| 生产模式 | 适用场景 | 核心规则 |
|---------|---------|---------|
| 单集手动逐步编辑 | 精品短剧精细化打磨 | 按步骤单环节扣除积分，可逐集、逐分镜编辑确认，可控性强 |
| 完整剧集自动创作 | 批量短剧高效产出 | 一次性扣除全剧创作所需积分，离线自动生成，创作过程中不可编辑，可在「作品管理」查看进度 |

#### 分镜精细化编辑规范

**合规红线**：
- 单个分镜最多 **1个出镜场景、5个出镜角色**
- 至少保留 **1个角色/场景**
- 避免违规导致生成失败、积分浪费

**核心参数编辑**：
- 出镜角色/场景
- 画面景别/视角
- 画面描述
- 台词音色/情感/内容
- 视频运镜
- 视频描述
- 支持删除冗余分镜

**双生成模式选择**：

| 模式 | 说明 |
|-----|------|
| 提示词描述模式 | 适合创意化自由创作 |
| 上传首尾帧模式 | 适合精准控制画面起终点 |

⚠️ 上传图片需＜5M，高清原图生成效果更佳；尾帧可直接选用下一分镜首帧，保证镜头衔接流畅。

### 步骤5：分镜画面环节

#### 图片模型选型规则

| 模型 | 特点 | 适用场景 |
|-----|------|---------|
| 图片1.0（推荐） | 人物一致性强，画风多元且稳定 | 绝大多数短剧创作场景 |
| 图片2.0 | 人物一致性强，画面遵从度稳定 | 固定风格的长剧集创作 |
| 图片3.0 | 画风多元，但多人一致性较差 | 仅推荐单人特写镜头使用 |

#### 画面优化核心操作

- 不满意的分镜可修改prompt、更换参考图重绘
- 可调整景别、视角、画面描述
- 保证与全片人设、场景风格统一
- 支持单张/批量下载分镜画面、对应音频
- 可随时删除无效分镜

⚠️ **关键提醒**：本环节生成的分镜图片，将作为分镜视频环节的首帧图，需确认画面无误后，再进入视频生成环节。

### 步骤6：分镜视频环节（含ViduQ2专属技巧）

#### 视频模型选型与成本对比

| 视频模型 | 5s分镜积分消耗 | 核心优势 |
|---------|---------------|---------|
| Vidu Q2（推荐） | 11积分/分镜 | 指令遵循能力强，大幅降低修改次数；角色动作流畅、大运镜能力突出；多图参考生成稳定性强；同效果下成本更低 |
| seedance 1.0 pro | 20积分/分镜 | 多镜头切镜能力强，运镜方式多样 |
| 视频2.0 | 7积分/分镜 | 动作稳定性强，模型成本低，适合批量粗剪 |

**ViduQ2模型选择路径**：
1. 平台web首页，上传剧本/选择小说后，在下方tab栏直接选定
2. 分镜视频环节，下拉切换视频模型为ViduQ2

#### ViduQ2视频生成方式

**方式一：参考生视频（推荐）**

平台自动拆分N个分镜，自动应用参考角色、参考场景，给出台词内容和视频描述。

**画面增强功能**：

打开"画面增强"按钮后，模型会先生成一张关键帧参考图，再参考这张图生成视频

- **无画面增强**：模型仅参考角色图和场景图生成视频
- **有画面增强**：模型先生成关键帧（参考角色+场景+画面描述词），再参考关键帧生成视频

**画面增强效果**：
- 加强生成结果的稳定性
- 更好解决人物与场景的融合问题
- 满足作者具体的想象

**画面增强使用技巧**：
- 画面描述与视频描述在景别、主体、动作等元素上需要保持较强的一致性
- **画面增强删除**：在分镜视频阶段，右侧"参考图片"处删除关键帧参考图，点击"重绘视频"
- **关键帧参考图替换**：可通过其他AI平台生成新图片替换，但需与平台现有风格保持一致

**方式二：首尾帧生视频**

**分镜规划阶段**：
- ✅ 推荐：首尾帧都通过本地上传（风格一致性更好）
  - 可在角色场景阶段下载人物三视图+场景图
  - 可下载分镜视频已生成的视频截图参考生成
- ❌ 不推荐：首帧通过提示词生成+尾帧空白不上传（效果不稳定）

**分镜视频阶段**：
- 首尾帧仅支持本地上传
- 尾帧图可选择下一个分镜的首帧
- 可自由选择"参考生视频"或"首尾帧生视频"
- 可编辑参考图、视频描述和台词内容进行多次抽卡调优

**使用建议**：在分镜规划阶段主要采用"参考生视频"，在分镜视频阶段，如需加强物体运动可控性或人物表情细节控制，可尝试首尾帧生成优化效果。

#### 视频精细化优化操作

- **重绘视频**：修改视频prompt精准调整运镜、人物动作、画面内容
- **进阶功能**：支持对口型、AI音效添加、自定义尾帧图上传
- **素材导出**：支持单条/批量下载有字幕/无字幕版本的分镜视频

### 步骤7：视频合成最终环节

#### 成品配置技巧

- **背景音乐**：支持平台音乐库匹配适配BGM，也可本地上传定制音乐
- **基础设置**：可自定义视频标题是否显示、字幕是否开启
- **完成**：点击「确认并合成视频」，完成全片制作

#### 作品后续管理

合成完成后，可在「作品管理」界面：
- 查看成片
- 二次编辑内容
- 追加剧集
- 持续优化作品

## 三、Prompt撰写核心技能

### 分镜画面重绘Prompt标准公式

**固定格式**：
```
画面风格 + 场景（参考图X） + 角色（参考图X） + 动作/神态 + 环境/时间 + 景别视角
```

**示例**：
```
中国现代动漫风格：枫叶林（参考图2），江眠（参考图1）在右侧举着书戳向左侧的林舟（参考图3），林舟回头看向江眠，眼神错愕，白天，中景平视镜头
```

**核心要点**：
- 严格锁定人物对应关系
- 动作指令清晰简洁
- 不随意更改初始画面风格
- 保证全片人物、画风一致性

### 分镜视频重绘Prompt标准公式

**固定格式**：
```
镜头运镜方式 + 角色动作序列 + 画面细节 + 口型要求
```

**示例**：
```
俯拍镜头，[图片中的女性]先蹲在落叶堆里翻找物品，[图片中的男性]在一旁驻足观看后同步蹲下，两人一同在落叶中寻找，动作流畅自然，视频中人物保持沉默，嘴部紧闭
```

**核心要点**：
- 运镜方式与提示词强对应
- 动作指令拆解清晰
- 保证画面连贯性
- 对口型场景需明确标注台词匹配要求

### ViduQ2视频提示词技巧

**提示词公式**：
```
主体/场景 → 场景描述 → 环境描述 → 艺术风格/媒介
```

**撰写要点**：

1. **调整句式和语序** - 避免主体物过多/复杂，主体物分散的句式描述
   - ✅ "李斌打了刘建树一巴掌"
   - ❌ "李斌的右手前手掌打在刘建树左脸的颧骨处"

2. **避免模糊术语** - 尽可能准确
   - ✅ "人物占据画面仅1/4的画幅"
   - ❌ "人物占据画面比较小的位置"

3. **使用口语化措辞** - 避免过度文学化叙述
   - ❌ 避免"李斌宛如流放的千古文人一般孤独"

4. **丰富准确完整** - 才能生成特定艺术风格、满足需求的视频

| 维度 | 关键提示词 |
|-----|-----------|
| **景别** | 大远景、远景、全景、中景、近景、特写、大特写 |
| **视角** | 低角度、高角度、平视镜头、鸟瞰镜头、正面拍摄、侧面拍摄、背面拍摄、俯视镜头、仰视镜头 |

| **运镜** | **固定**：镜头固定、Static shot、固定镜头等<br>**推**：镜头推进、镜头zoom in、推镜头等<br>**拉**：镜头拉远、镜头zoom out、拉镜头等<br>**摇**：镜头向左摇、摇镜头<br>**左移**：镜头向左移动、镜头move left、左移镜头、镜头track left等<br>**右移**：镜头向右移动、镜头move right、右移镜头、镜头track right等<br>**跟**：镜头跟着人物向前/后/左/右运动<br>**升**：镜头向上移动、镜头move up、升镜头等<br>**降**：镜头向下移动、镜头move down、降镜头等 |

| **特效** | **爆炸**：爆炸特效、Explodes<br>**旋涡**：旋涡特效、Vortex<br>**粉碎**：粉碎特效、shatters<br>**扭曲**：扭曲特效、Warps<br>**融化**：融化特效、Melt<br>**消失**：变成气泡消失了、变成雪花消失了等<br>**石化**：变成石头 |
| **氛围** | **快乐**：明亮色彩<br>**悲伤**：冷色调<br>**宁静**：柔和色调<br>**紧张**：阴暗的光线<br>**温馨浪漫**：暖色调、柔和光线 |

### 提示词调优方法

#### 方法一：AI辅助撰写提示词

如果人工无法写清句子的主谓宾结构，可与AI沟通，让AI给出简单合适的提示词。

**示例**：
- **基础提示词**："李斌坐在办公室加班，脸上痛苦的神情"
- **AI优化后**："中近景，职场男性李斌坐在现代办公室的办公桌前深夜加班，面前电脑亮屏、桌上摆着零散文件；人物神态：眉头紧皱，眼神黯淡疲惫，脸颊微沉，手部轻抵额头，呈现出身心俱疲的痛苦与烦躁，画面色调偏冷，环境安静无多余人物"

#### 方法二：适度"联想"优化

第一次提示词可能很简单，AI无法生成理想细节，可在优化时进行适度联想扩充。

**示例**：
- **基础提示词**："玻璃桌上的咖啡杯，杯子外面写着单词LOVE"
- **联想优化**："都市漫画风格，花园里的透明玻璃桌上的咖啡杯，杯子外面写着单词LOVE，周围满是盛开的鲜花，和煦的阳光洒满整个花园"

⚠️ **注意**：如果多次修改提示词，有些效果无论怎么修改都无法达到，可能是模型本身的效果限制。

## 五、避坑与合规操作

### 核心操作避坑点

| 避坑点 | 说明 |
|-------|------|
| 剧本大纲确认后不可回退 | 角色生成前务必核对所有内容，避免大规模返工 |
| 单分镜角色/场景数量规则 | 严格遵守：最多1个场景、5个角色，至少保留1个角色/场景 |
| 上传参考图大小限制 | 需控制在5M以内，优先使用高清原图，模糊图易导致画面崩坏 |
| 分镜画面是视频首帧 | 需确认无误后再进入视频环节，避免后续反复重绘消耗积分 |

- 需自行对AI生成内容进行合规审核

## 七、常见问题

**Q: 上传文件格式不支持？**
A: 小说仅支持txt，剧本支持docx和txt格式

**Q: 角色生成后想修改剧本大纲？**
A: 角色生成后剧本大纲不支持回退，请确认后再继续

**Q: 如何获得更好的生成效果？**
A:
- 使用"画面增强"功能
- 上传清晰图片（<5M）
- 撰写清晰的prompt
- 保持人物名称和风格一致
- 画面描述与视频描述保持一致性
- 优先选用ViduQ2模型

**Q: 分镜画面和分镜视频的关系？**
A: 分镜画面生成的图片会作为分镜视频的首帧图

**Q: 参考生视频和首尾帧生视频如何选择？**
A: 推荐默认使用参考生视频。如需加强物体运动可控性或人物表情细节控制，可尝试首尾帧生成。

**Q: 如何降低积分消耗？**
A:
- 优先选用ViduQ2模型（比seedance1.0 pro节省45%积分）
- 前期确认好参数再启动生成
- 复用成熟素材，减少重复创作
