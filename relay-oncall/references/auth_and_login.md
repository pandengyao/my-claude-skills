## 认证方式一览

Relay 支持 4 种认证方式：指纹、扫码、短信验证码、动态令牌。

| 认证方式 | 适用场景 | 依赖条件 |
|----------|----------|----------|
| 指纹 | Mac端设备 | 需安装度管家 |
| 扫码 | 所有设备 | 手机如流APP |
| 短信验证码 | 所有设备 | 手机接收短信 / 如流服务号 |
| 动态令牌 | 终端SSH登录 | 手机如流-工作流-动态令牌 |

> 注意：账密认证已于2024年8月30日下线，替换为「动态令牌+如流推送」。

## 密码/账号类问题

### 现象：登录Relay出现passwd authentication failed等密码或账号报错
- **排查**：登录 email.baidu.com 检查邮箱密码是否正确；校验手机如流是否打开，手机时间是否正确
- **密码找回**：https://eac.baidu-int.com/app-account/pwd#/pwd/reset
- **注意**：Relay 密码与 uuap 密码一致

### 现象：手机认证失败，提示「请求已经过期，请在电脑上重新发起认证请求」
- **原因**：手机本地时间被调整
- **解决**：将手机系统时间设置为自动获取

### 现象：从Relay登陆服务器出现 Permission denied, please try again
- **原因**：服务器密码不正确，或无门神免密权限
- **解决**：联系部门OP处理，或申请门神权限免密登录
- **申请地址**：http://giano.baidu.com/dgweb/index.php?r=Privilege/PersonalInfo/DoorGodApply&nodeId=35465

## Beep认证问题

### 现象：beep认证失败
- **自查**：访问 https://apigo.baidu.com/d/8R3mLeZP，点击右上角「刷新认证记录」按钮，查询半小时内的认证失败原因
- **其他原因**：用户 ERP 账户被锁定（可能因更换设备、旧设备未格式化导致）。尝试退出 family 网站重新登录
- **升级**：beep 认证问题可寻求 EAC 团队帮助

### 现象：打开WebRelay没有弹出扫码认证或指纹认证
- **原因**：指纹认证依赖度管家认证服务
- **解决**：尝试重启度管家；如果无法解决请加入WebRelay&Relay用户4群：10198644 直接提问

### 现象：打开WebRelay认证时选择验证码选项提示关注服务号
- **解决**：关注「企业消息平台」服务号

### 现象：终端登录Relay，手机不弹beep认证且查不到beep认证失败记录
- **解决**：检查用户名是否拼错；使用 `ssh relay.baidu-int.com` 登录（外包同学请用 web 版）；或使用 webrelay https://webrelay.baidu-int.com/#/

## 指纹认证问题

### 现象：WebRelay无法使用指纹认证
- **前提条件**：指纹认证仅支持 Mac 端设备，且需要安装度管家
- **排查步骤**：
  1. **检查度管家安装及指纹注册**
     - 点击屏幕右上角度管家小图标
     - 点击度管家 app 右上角的设置按钮
     - 检查是否已开启指纹登录（如显示「解绑」，说明已注册；如显示「开启指纹登录」，需要先注册）
     - 如未注册，按照引导完成指纹录入（轻触 Touch ID 传感器）
  2. **检查度管家服务状态**
     - 尝试重启度管家应用
     - 如果重启后仍无法使用，请加入WebRelay&Relay用户4群：10198644 直接提问
  3. **检查本地代理设置**
     - 指纹调起失败时，检查本地是否开启了网络代理
     - 如有代理，尝试临时关闭后重试
  4. **验证指纹注册信息**
     - 如提示「1002:注册信息异常」或「注册信息异常，指纹认证失败」
     - 访问注册地址重新绑定：https://apigo.baidu.com/d/yvSVESiQ
     - 完成后在度管家中重新绑定指纹
  5. **尝试替代认证方式**
     - 如指纹识别持续失败（如冬季手指蜕皮），可使用扫码认证
     - 打开 WebRelay 时选择「扫码认证」或「验证码认证」
- **参考文档**：[手把手教您开启EAC指纹认证](https://ku.baidu-int.com/knowledge/HFVrC7hq1Q/HXFtYvbMQj/TNoZ2M6o-t/_vD-S2DoqtbtBl)

### 现象：relay-cli指纹识别失败
- **季节原因**：冬季手指蜕皮导致识别失败，可尝试扫码认证 `relay-cli -t qc`
- **补充**：尝试多录入几个指纹信息

### 现象：relay-cli指纹登录报错 unexpected end of JSON input
- **原因**：安全加固后旧版本不兼容
- **解决**：重新下载最新安装包

### 现象：提示注册信息异常，指纹认证失败
- **解决**：重新绑定度管家指纹

### 现象：invalid totp token
- **解决**：重启度管家

### 现象：指纹认证报错 connection refused
- **报错示例**：`Post "https://duguanjia-local.internal.baidu.com:50001/eac/fingerprint/auth": dial tcp 127.0.0.1:50001: connect: connection refused`
- **原因**：度管家服务未启动或连接失败
- **解决**：重启度管家应用

## 动态令牌问题

### 现象：输入令牌后没有反应，等一会儿后提示登录失败
- **解决**：尝试重启如流应用后重新登录

### 获取动态令牌方法
- 手机如流 → 工作流 → 应用 → 动态令牌

### 手机忘带了如何登录Relay
- PC端打开如流「工作流」，选择临时Token申请
- 登录Relay时把Token输入到密码框后回车即可

## 新员工/特殊身份登录

### 现象：入职第一天登录Relay失败
- **原因**：Relay用户信息每晚12点同步新同学账号
- **解决**：当晚12点后再尝试登录

### 分公司如何申请登录Relay
- 向安全部 anquan@baidu.com 发邮件申请，说明登录原因及申请人邮箱，抄送 chongbo02、liulu40

### 现象：No user exists for uid xxxxxx
- **解决**：重启客户端或客户端机器
