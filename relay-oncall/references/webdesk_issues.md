## 基础信息

- **支持协议**：RDP（Windows远程桌面）、VNC
- **体验地址**：webrelay.baidu-int.com/desk
- **WebDesk服务器IP**（白名单配置用）：10.21.2.128、10.21.244.58、10.45.33.111
- **RDP默认端口**：3389
- **VNC默认端口**：5900 + 显示ID（如 vncserver :14 → 端口 5914）

## 连接与错误码

### 错误码 519 - 上游服务器不存在
- **常见原因**：
  1. 机器未开白，WebDesk服务器到目标服务器RDP 3389端口不通 → 联系相关同学对WebDesk服务器IP加白
  2. 许可证到期 → 参考 https://cloud.tencent.com/document/product/213/41509
  3. 机器设置了超时退出，自动踢掉会话 → 查看事件日志分析
- **排查**：检查对应账户是否具有远程登录权限（加入 remote desktop users 组）

### 错误码 514 (VNC)
- **常见原因**：密码错误或vncserver显示ID不对

### 错误码 769 - 用户未登录
- **一般原因**：密码错误
- **域账号登录**：账户名为邮箱，密码为邮箱密码（通过 mail.baidu.com 验证）

### 计算机域账号禁用问题
- **现象**：本地可登录，WebDesk连接报519或769
- **原因**：机器退域后该域账号无法远程登录
- **解决**：创建本地账户迁移配置，或联系ithelp重新加入域

## 剪切板问题

### 剪切板不同步
- WebDesk会自动双向同步剪切板，如不同步多试几次
- **排查**：
  1. 确认浏览器给了剪切板权限
  2. macOS用 command+c，Windows用 ctrl+c
  3. 尝试重启远程机器的RDP剪切板进程

## 输入法与按键问题

### 输入中文
- WebDesk只向服务器发送按键信息，输入中文需目标服务器安装中文输入法且处于中文模式

### macOS按键映射
- Command → Ctrl，Ctrl → Windows（仅macOS生效）
- 修改方法：添加/修改服务器菜单中打开映射选项

### Shift键输入不符合预期
- **原因**：切换到后台后shift键未释放
- **解决**：单独按下再抬起shift键解除触发状态

### macOS快捷键无法触发
- **原因**：系统级快捷键优先级更高（如 command+tab）；option键触发特殊字符

## 画面显示问题

### VNC画面有黑边
- **解决**：调节vncserver启动时的画面尺寸参数

### VNC画面跟着鼠标移动（突然放大）
- **解决**：Alt + 鼠标滑轮进行缩放还原

### 像素密度设置
- 默认值96，2K屏设为192（96*2），4K屏设为384（96*4）

## 文件传输
- **仅支持**标准Windows RDP服务器（不支持xrdp、VNC）
- 使用 Web Desk Driver 网络磁盘进行传输
- **注意**：临时磁盘每次连接刷新，不要在其中保存文件

### 上传步骤
1. 通过网页把本地文件上传至WebDesk网络磁盘
2. 在服务器中把网络磁盘中的文件保存至服务器磁盘

### 下载步骤
1. 在服务器中把需要下载的文件拖至WebDesk网络磁盘
2. 在网页中下载

## 其他问题

### 连接到服务器很卡
- WebDesk本身通常无故障，检查服务器是否高负载

### Edge浏览器手势干扰
- 参考微软官方文档关闭鼠标右键手势

### 添加macOS远程桌面
- 协议：VNC，端口：5900，用户名密码为macOS系统账号（非VNC密码）
