**成功响应** (HTTP 200):

| 参数名 | 类型 | 说明 | 示例值 |
|-------|------|------|--------|

| msg | string | 响应消息 | "OK" |

```json
{
  "code": 200,
  "msg": "OK",
  "data": {
    "id": "xxx_123456",
    "name": "示例名称",
    "createTime": "2026-03-03 10:00:00"
  }
}
```

| 错误码 | HTTP状态码 | 错误消息 | 触发条件 | 处理建议 |

| 40002 | 400 | "参数格式错误" | param1 格式不正确 | 提示正确格式 |
| 40301 | 403 | "无操作权限" | 用户无权限 | 提示申请权限 |
| 50001 | 500 | "系统错误" | 服务器异常 | 稍后重试 |
