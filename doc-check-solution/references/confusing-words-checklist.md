# 易混淆词对检查列表

## 同音/近音词对

| 错误词 | 正确词 | 区别说明 | 示例 |
|--------|--------|----------|------|
| 如果 | 如何 | "如果"表示假设条件；"如何"表示方式方法 | 因此如何从国外框架迁移 → 而非因此如果 |

**语义判断方法**：
- **如果**：用于假设条件，后面通常接"就/则/那么"等表示结果
  - 正确示例：如果采用这种方法，就能提高效率
  - 错误示例：因此如果从国外框架迁移便成为难点（语义不通）
- **如何**：用于询问或陈述方式方法，后面通常接动词短语
  - 正确示例：如何从国外框架迁移成为关键难点
  - 正确示例：我们讨论如何解决这个问题
- **判断技巧**：看句子是在陈述假设条件，还是在陈述问题/方法
| 截止 | 截至 | "截止"是动词；"截至"是介词，用于时间范围 | 截至2025年7月31日 → 而非截止 |
| 必须 | 必需 | "必须"是副词，表示一定要；"必需"是形容词，表示不可缺少的 | 必须完成这项任务；必需的设备 |
| 启示 | 启事 | "启示"是启发提示；"启事"是公开声明 | 得到重要启示；发布招聘启事 |
| 账号 | 帐号 | "账号"是用户账户；"帐号"是账户号码 | 登录账号；银行帐号 |
| 反应 | 反映 | "反应"是机体受刺激后的活动；"反映"是客观事物表现 | 身体反应；反映情况 |
| 制定 | 制订 | "制定"是确定；"制订"是创制拟定 | 制定计划；制订方案 |
| 权利 | 权力 | "权利"是公民依法享有的利益；"权力"是政治上的强制力量 | 维护权利；行使权力 |
| 以致 | 以至 | "以致"表示结果；"以至"表示时间、程度、范围的延伸 | 以致失败；以至今天 |

## 语义易混淆词对

| 错误词 | 正确词 | 区别说明 | 示例 |
|--------|--------|----------|------|
| 框架 | 格式 | 深度学习框架 vs 模型格式 | PyTorch是框架；ONNX是格式 |
| 训练 | 推理 | 模型学习 vs 模型预测 | 训练模型；推理预测 |
| 参数 | 超参数 | 模型权重 vs 训练配置 | 模型参数；学习率超参数 |
| 编译器 | 解释器 | 编译型语言 vs 解释型语言 | C++编译器；Python解释器 |
| GPU | 显卡 | 芯片 vs 完整硬件 | GPU芯片；显卡硬件 |
| 库 | 框架 | 工具集 vs 完整平台 | Transformers库；PyTorch框架 |
| 算子 | 操作 | 深度学习算子 vs 通用操作 | 卷积算子；数据库操作 |
| 张量 | 矩阵 | 多维数组 vs 二维数组 | 3D张量；2D矩阵 |

## 检查方法

1. **上下文分析**：结合前后文语境判断词义
2. **语义匹配**：检查词与词之间的语义搭配是否合理
3. **专业验证**：对于专业术语，对照知识库验证准确性
4. **逻辑验证**：检查句子逻辑是否通顺，用词是否准确

## 使用建议

- 检查时逐句扫描，特别关注易混淆词对
- 对于不确定的用词，查阅词典或专业资料
- 建立个人易错词本，定期复习
- 重要文档建议多人交叉检查
